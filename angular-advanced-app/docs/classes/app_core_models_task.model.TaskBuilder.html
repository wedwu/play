    Class TaskBuilder

    Fluent builder for constructing validated Task instances. All setter methods return this for chaining.

    Example

    const task = new TaskBuilder('Fix login bug')
      .withPriority(TaskPriority.HIGH)
      .withAssignee('user-123')
      .withDueDate(new Date('2025-01-31'))
      .build();
