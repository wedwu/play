    Class DashboardComponent

    Main dashboard page component.

    Displays aggregate task statistics, urgent tasks, a mini Kanban board overview, and a team member list. Demonstrates all major Angular lifecycle hooks with real-time logging in a debug panel.

    Uses OnPush change detection and exposes observables for async pipe usage.
